# dame bash

## Install Guide

1. clone the repo

```bash
git clone
```

2. install dame cli

```bash
sh install.sh
```

This will install scripts

## Update Guide

1. pull latest changes

```bash
git pull
```

2. reinstall dame cli

```bash
sh install.sh
```

This will just reinstall the scripts folder

<br>

# Usage

Type dame in your terminal and see the available scripts
